# netflix-eda
This project is an exploratory data analysis of the Netflix movies and TV Shows dataset sourced from kaggle. I used Python, Pandas, Seaborn, and Matplotlib to clean the data, explore content trends over time, and visualize patterns across genres, ratings, and countries. The most interesting finding was that Netflix's content library exploded starting in 2016 as they pushed into interntional markets with Interntional Movies being the most common genre on the platform. This provides insight into Netflix's strategy being on the global market not just American.
